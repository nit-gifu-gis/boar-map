## 使用言語/フレームワーク

- TypeScript
- Node.js v20.00 >=
- Pnpm
- Next.js v15
- Tailwindcss v3.4.1

## コマンド

```bash
# 依存関係インストール
$ pnpm install

# パッケージの追加 (本番環境に必要なもの)
$ pnpm add <パッケージ名>

# パッケージの追加 (開発環境に必要なもの)
$ pnpm add -D <パッケージ名>

# 開発用サーバー起動
$ pnpm dev

# ビルド + サーバー起動
$ pnpm build
$ pnpm start
```
